# AwesomeApp

This is an simple app React Native Hooks with Dark Mode function.

## Installation

Use the package manager npm / yarn to install package.

```bash
cd AwesomeApp && yarn install
or
cd AwesomeApp && npm install
```

## Usage

```bash
$ react-native run-android
```
